5.3 Getting the second flag



1. On the “reverse shell” run the following commands.

$cd root
$ls
$cat horcrux2.txt


Output:


The part in the curly brackets (MjogbWFSdm9MbyBHYVVudCdzIHJpTmcgZGVTdHJPeWVkIGJZIERVbWJsZWRPcmU=) is a base64 encoded string.

2. Let's decode it. On your Kali Machine run the following command.

$echo 'MjogbWFSdm9MbyBHYVVudCdzIHJpTmcgZGVTdHJPeWVkIGJZIERVbWJsZWRPcmU=' | base64 --decode
