3. Enumerate



If you visit the 192.168.12.19 URL you got nothing interesting.

a) Enumerate information about the directories.

$dirb http://192.168.12.19


Output:



This gave us a directory called “blog”.

If you enter to this directory you can see the links point out to “wordpress.aragog.hogwarts”.

b) Let's add this direction to “/etc/hosts” file in order to access directly.

ON your Kali Machine add the following line to the file and save it:
192.168.12.19 wordpress.aragog.hogwarts


b) Let's open the URL.



So, it's a Wordpress site. Time to fire up “wpscan”, and because of that message about plugins we will set plugin detection to aggressive.


c) Let's run “wpscan”.

$wpscan --url http://192.168.12.19/blog/ --plugins-detection aggressive  --api-token  <your token from www.wpscan.com/profile>
