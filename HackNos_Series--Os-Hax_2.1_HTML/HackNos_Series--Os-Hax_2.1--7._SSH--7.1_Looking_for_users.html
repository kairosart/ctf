7.1 Looking for users



Let's see the home folder.

$ ls home


Output:
rohit

So there's an user called “rohit”.

There should be a file containing the hash for the rohit user password. Let's look for it.
After looking around different folders you'll find a “backups” folder in the “var”

$ grep -iRl "rohit" /var/backups


Output:
/var/backups/passbkp/md5-hash

That's our file.
