4.3.1 Steganography



Let's extract text from the “vishal.jpg” file with “steganography”.

1. Install “stegcracker” on your Kali Machine.
$sudo apt-get install steghide -y


2. Run the program.
$stegcracker vishal.jpg /usr/share/wordlists/rockyou.txt


Output:


The string “iloveyou” was used to write something to the file. You can find the written data on the“vishal.jpg.out” file.

3. Show the content of the“vishal.jpg.out” file.
$cat vishal.jpg.out


Output:
