We'll try to identify SUID enabled binaries with the help of find command.
With this command the machine will display the full paths to all the accessible files on the computer, without displaying the "Permission denied" errors that are likely to occur.

$ find / -perm -u=s -type f 2>/dev/null


We've found “/script/find” has SUID permissions and it works similarly as Linux-Find utility thus we try to execute /bin/sh command and obtained access of the nightfall shell.
