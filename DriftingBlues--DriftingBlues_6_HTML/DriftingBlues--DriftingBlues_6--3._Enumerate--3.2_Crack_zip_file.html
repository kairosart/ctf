3.2 Crack zip file



You don't have the password for the “spammer.zip” file, but you can crack it with the “fcrackzip” tool.

1. Run “fcrackzip”.
$fcrackzip -u -D -p /usr/share/wordlists/rockyou.txt spammer.zip


Output:


The passoword is myspace4.

2. Extract the files.
$unzip spammer.zip


Output:


3. Show the file “credit.txt”.
$cat creds.txt


Output:


Those are the credentials to log in “Textpattern”.
