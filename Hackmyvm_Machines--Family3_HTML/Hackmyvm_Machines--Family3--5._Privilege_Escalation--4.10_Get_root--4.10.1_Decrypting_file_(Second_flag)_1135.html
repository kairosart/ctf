4.10.1 Decrypting file (Second flag)


1. Look what kind of file is “root.txt”.

root@family:~# file root.txt

Output:

So you need a password to decrypt the file.

2. “lsblk” lists information about all available or the specified block devices. 

root@family:~# lsblk

Output:

3. Mount a device.

root@family:~# mount /dev/sda3 /mnt
root@family:/mnt# ls

Output:

4. Show the password file.

root@family:/mnt# cat password

Output:

5. Copy this password to “root" directory.

root@family:/mnt# echo "QHSvtnwvnUgKRGDQfG6rC58bAU4woNIW0Z7eL6ma" > /root/pass

This password will let you to decrypt the “root.txt” file.

6. Figure out the Openssl veersion.

root@family:/mnt# openssl version

Output:

7. Decrypt "root.txt' file. You have a password from point 5.

From https://www.openssl.org/docs/man1.1.1/man1/openssl-enc.html

root@family:~# cat pass

QHSvtnwvnUgKRGDQfG6rC58bAU4woNIW0Z7eL6ma

root@family:~# openssl enc -aes128 -pbkdf2 -d -in root.txt -out file.txt

Enter the password when requested.

8. Get the flag.

root@family:~# cat file.txt

Output:

Index
