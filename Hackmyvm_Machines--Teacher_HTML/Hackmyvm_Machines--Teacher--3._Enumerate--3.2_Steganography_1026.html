3.2 Steganography


Steganography is the practice of concealing a message within another message or a physical object. In computing/electronic contexts, a computer file, message, image, or video is concealed within another file, message, image, or video.

1. On your Kali Machine download and follow the instructions to install the “stegseek” application from the github https://github.com/RickdeJager/stegseek.

$ sudo apt install ./stegseek_0.6-1.deb

2. Download the “rabbit.jpg” file from the victim's machine.

$ wget http://192.168.12.65/rabbit.jpg

3. Run the following code.

$ stegseek rabbit.jpg

Output:

You'll get a file called “first_Logo.jpg.out”.

4. Show the contente of the “first_Logo.jpg.out” file.

$ cat first_Logo.jpg.out

Output:

Index
