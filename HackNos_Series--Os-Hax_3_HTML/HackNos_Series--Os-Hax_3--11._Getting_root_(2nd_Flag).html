11. Getting root (2nd Flag)



“sudo su” will put you into a root environment but it will ask you for your user password instead of the root password (once sudo has given you root privileges, “su -” can be executed with no password).

blackdevil@hacknos:~$ sudo su


Output:
root@hacknos:/home/blackdevil#

You're root.


Let's go to “root” directory.

root@hacknos:/home/blackdevil# cd /root
root@hacknos:/home/blackdevil# ls


Output:
root.txt snap


Let's see the second flag.

$cat root.txt


Output:

Index
