4.2 Creating a Debian package



To create a new Debian package we're going to use “fpm”. You can get more info here.

a) Install “fpm” on your Kali Machine if you don't already have it.

$ apt-get install ruby ruby-dev rubygems build-essential
$ gem install --no-document fpm


b) Create a file called “shell.sh”. You can make a new directory to work.

$ mkdir shellsym
$ cd shellsym/
$ nano shell.sh


c) Add these lines to “shell.py”.

#!/bin/bash
/bin/bash


d) Generate a Debian package for “shell.sh”.

$ fpm -s dir -t deb -n shellsym --before-install shell.sh ./


Output:
Created package {:path=>"shellsym_1.0_amd64.deb"}

e) List the files.

$ ls


Output:

Index
