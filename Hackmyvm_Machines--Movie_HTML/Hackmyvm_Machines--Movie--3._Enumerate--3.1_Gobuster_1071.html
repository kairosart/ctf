3.1 Gobuster


1. Run “gobuster” to enumerate the victim's machine.

$ gobuster -q dir -w /usr/share/seclists/Discovery/Web-Content/common.txt -u http://192.168.12.69 -x php

Output:

There are two interesting directoriesl

2. Run “gobuster” again to enumerate “/data” directory.

$ gobuster -q dir -w /usr/share/seclists/Discovery/Web-Content/common.txt -u http://192.168.12.69/data -x php

Output:

The file config.php seems interesting.
