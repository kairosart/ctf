Install the machine on VirtualBox:

1. Create a new folder in your virtual hard disks repository.
$ mkdir /home/<your username>/<your repository>/VulnOSv2 

2. Extract the file downloaded to that folder.
3. On Virtualbox choose Tools->Add.
4. Go to the folder VulnOS and select the file “VulnOS.vbox”.
5. Open the new Machine on Virtualbox.

Virtual Machine Network Settings

Both Machines have to be on “Bridged Adpapter”.
