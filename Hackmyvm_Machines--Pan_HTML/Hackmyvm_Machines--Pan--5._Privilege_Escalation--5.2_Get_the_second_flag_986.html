5.2 Get the second flag


1. Show the flag.

root@pam:/# cd /root
root@pam:/# ls
root@pam:/# cat root.enc

Output:

It seems an encrypted text.

2. You have to figure out what kind of encryption is.

root@pam:~# file root.enc

Output:

It's an “Openssl” encrypted text.

3. On the victim's machine run the following command to decrypt the file “root.enc”.

root@pam:/var/www/html# openssl enc -aes-256-cbc -d -in root.enc -out /tmp/root.txt

You'll be asked for a password. Introduce “rootisCLOSE”.

Note

Don't worry if you get an error like this.

4. Show the content of “/tmp/root.txt”.

root@pam:/var/www/html# cat /tmp/root.txt

Output:

Index
